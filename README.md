# Upscalingway

Automatically sets upscaling to FSR in the main menu (to allow render resolution scaling) and switches to DLSS on login (to force custom resolutions from DLSS Tweaker). This effectively "unsticks" DLSS from DLAA.

## Installation

Add the following Custom Repository URL to your Dalamud Plugin Installer settings:
```
https://raw.githubusercontent.com/KuroZantetsuken/Upscalingway/main/pluginmaster.json
```
